# 주사위 골목 🎲

브라우저로 즐기는 주사위 게임 모음. 카드 하나를 고르면 바로 플레이 — 로컬/AI 대전, 온라인 멀티플레이, 투명 데스크톱 앱까지. 설치 없이 링크 하나로 친구를 부른다.

> 골목에 모인 다섯 가지 게임: **윷놀이 · 요트 다이스 · 너클본즈 · 라이어 다이스 · 좌·중·우(LCR)**

---

## 🎮 게임 5종

### 🎋 윷놀이
넉 장의 윷을 던져 말 4개를 먼저 완주시키는 한국 전통 보드게임. 도·개·걸·윷·모에 빽도까지, 상대 말을 잡으면 한 번 더, 말을 업어 함께 이동. **파워 게이지**를 꾹 눌렀다 떼는 타이밍으로 윷·모 확률을 올리고, 꼭짓점에서 지름길 판단, 랜덤 **늪(구렁텅이)** 함정까지. 2~6인 개인전 / 2:2 팀전, 온라인에선 로비에서 **팀 직접 선택**과 **관전 모드** 지원. AI 난이도 3단계.

### 🎲 요트 다이스
5개의 주사위를 굴려 12개 칸을 채우는 고전 점수 게임. 규칙 3종 지원:
- **요트 다이스** (국내 앱 규칙): 포카드·풀하우스 = 주사위 5개 합, 상단 63점↑ 보너스 +35
- **야찌** (Yahtzee): 풀하우스 25 / 스트레이트 30·40 고정, 트리플 추가, 보너스 +35
- **오리지널** (1938): 포카드 = 4개 합, 스트레이트 정확히 30점, 보너스 없음

2~6인, AI 난이도 3단계(몬테카를로 킵 판단 + 상단 보너스·칸 희소가치 고려).

### 🎲 너클본즈 (Knucklebones)
3×3 격자를 주사위로 채우는 1:1 두뇌 대결. 한 열에 같은 숫자를 모으면 점수가 곱으로 폭증(3·3 = 12점, 3·3·3 = 27점). 내가 놓은 수와 같은 수가 상대의 같은 열에 있으면 **전부 파괴**. 격자가 꽉 차면 총점 승부.

### 🤥 라이어 다이스 (Liar's Dice)
컵 속에 주사위 5개를 숨기고 전체 개수를 두고 뻥카를 치는 심리 게임. "전체에 4가 3개 이상" 식으로 베팅, 1은 만능(와일드). 다음 사람은 더 높게 올리거나 **거짓말(Dudo)**·**정확히(Calza)**로 도전. 주사위를 다 잃으면 탈락, 최후의 1인이 승리. 2~4인.

### 🔀 좌·중·우 (LCR)
칩 3개로 시작하는 순수 운빨 파티 게임. 가진 칩 수만큼 주사위를 굴려 **L**(왼쪽 주기)·**R**(오른쪽 주기)·**C**(가운데 팟)·**•**(유지). 마지막까지 칩이 남은 한 명이 팟까지 싹쓸이. 3~6인, 친구 여럿이 왁자지껄할 때 제맛.

## ✨ 공통 기능

- **온라인 멀티 (서버 권위형)**: 서버가 게임을 직접 운영해 주사위·턴·점수를 관리 — 결과 조작 불가. 방 코드/초대 링크(`?room=코드`)로 참여.
- **AI 봇 채우기**: 온라인 방에서 사람이 부족해도 호스트가 🤖 버튼으로 AI를 채워 바로 시작.
- **방장 자동 위임 & 견고성**: 호스트가 나가도 다음 사람이 자동으로 방장이 되고, 끊긴 자리는 자동 진행돼 게임이 멈추지 않음. 재매치도 그대로 유지.
- **재접속**: 새로고침하거나 잠깐 끊겨도 방 코드+자리를 기억해 복귀.
- **인터랙티브 튜토리얼**: 라이어·너클본즈·좌중우는 그림과 함께 단계별로 규칙을 익히고 바로 시작.
- **전적 · 연승 · 도전과제**: 게임별 승/패·승률·연승 기록, 뱃지 수집(첫승·연승·베테랑 등). 기기에 로컬 저장.
- **결과 자랑하기**: 게임 끝나면 결과를 공유 시트(모바일)나 클립보드로 — 연승도 함께 뽐냄.
- **이모지 반응 & 콤보**: 온라인에서 놀림 이모지·한마디 던지기, 같은 이모지 연타 시 콤보 배지.
- **5가지 카페 테마**: 에스프레소·민트초코·자몽·블루레몬·흑임자 (골목 전체 공유).
- **팀전 & 관전**: 윷놀이 2:2 팀전(로비에서 팀 직접 선택), 정원이 차면 늦게 온 친구도 **관전자**로 입장해 구경.
- **반응형 3단 레이아웃**: 넓은 화면은 3단 가로(양옆에 정보·컨트롤), 좁으면 2단→1단으로 자동 적응.
- **접근성**: 팀 구분을 색+무늬(점선)+아이콘으로 이중 표시(색맹 대응), 주요 요소 aria 라벨, 모바일 햅틱 진동.
- **창 투명도 조절**: 데스크톱 앱에서 창을 반투명하게 + 항상 위 고정/최소화 (게임별 우측 상단 슬라이더).
- **PWA**: 홈 화면에 설치해 앱처럼 실행. 인게임 음소거, 입·퇴장 토스트.

## 🚀 로컬 실행
```bash
npm install
npm start
# http://localhost:3000
```

## 📦 배포 (Render 등)
- Build Command: `npm install`
- Start Command: `node server.js`
- 정적 파일 + WebSocket을 한 서버에서 처리하므로 별도 설정 불필요

## 🖥️ 데스크톱 앱 (투명 창, 선택)
```bash
npm install
npm run app   # electron 창으로 실행 (항상 위, 투명 배경)
```

## 🗂️ 구조

허브(`index.html`)에서 각 게임으로 이동. 게임마다 엔진이 **독립 파일**(서버가 require)과 **HTML 내 인라인**(브라우저용)으로 이중 존재하며, 로직은 항상 동일하게 유지됩니다.

```
public/
├─ index.html        허브 "주사위 골목"
├─ yut.html          윷놀이        (yut-core.js 인라인 + ws)
├─ yacht.html        요트 다이스   (game-core.js 인라인)
├─ kb.html           너클본즈      (kb-core.js 인라인 + ws)
├─ ld.html           라이어 다이스 (ld-core.js 인라인 + ws)
├─ lcr.html          좌·중·우      (lcr-core.js 인라인 + ws)
├─ yut-core.js       윷놀이 엔진 (이동·잡기·팀전·늪·타이머)
├─ game-core.js      요트 엔진 (규칙·점수·AI)
├─ kb-core.js        너클본즈 엔진
├─ ld-core.js        라이어 엔진 (숨김정보·블러핑 AI)
├─ lcr-core.js       좌·중·우 엔진
├─ manifest.json · sw.js · icon-*.png   PWA
server.js            정적 서버 + 방/로비 관리 + 방마다 엔진 1개를 돌리는 권위 서버
electron/            투명 데스크톱 래퍼
```

- **서버 권위형**: 클라이언트는 상태 스냅샷을 받아 렌더만 담당. 라이어 다이스는 숨김정보 때문에 플레이어별로 다른 시점을 전송.
