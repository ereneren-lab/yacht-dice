# 요트 다이스 🎲

브라우저로 즐기는 요트 다이스. 로컬/AI, 온라인 멀티플레이, 투명 데스크톱 앱까지.

## 규칙 모드 3종
- **요트 다이스** (국내 앱 규칙): 포카드·풀하우스 = 주사위 5개 합, 상단 63점↑ 보너스 +35
- **야찌** (Yahtzee): 풀하우스 25 / 스트레이트 30·40 고정, 트리플 추가, 보너스 +35
- **오리지널** (1938): 포카드 = 4개 합, 스트레이트 정확히 30점, 보너스 없음

## 주요 기능
- 2~6인 로컬 핫시트 + **AI 난이도 3단계**(쉬움/보통/어려움). AI는 몬테카를로로 킵 판단하고, 상단 보너스·칸 희소가치를 고려해 점수를 기록함
- **온라인 멀티 (서버 권위형)**: 서버가 게임을 직접 운영해서 주사위·턴·점수를 관리. 호스트가 나가도 게임이 멈추지 않고, 결과 조작이 불가능함
- **재접속**: 새로고침하거나 잠깐 끊겨도 방 코드+자리를 기억해 원래 자리로 복귀
- 굴릴 때마다 15초 턴 타이머, 시간 초과 시 자동 마무리
- 점수 **기록 확인 창**(실수 방지), 인게임 **음소거**·나가기
- **전적 저장**(플레이 수/최고점/승리/요트 횟수) — 기기에 로컬 저장
- 주사위 소리(웹오디오), 창 투명도 조절, 키보드(Space/1~5), 펠트 트레이(킵/굴림 분리)
- **캐릭터 선택**(이모지 아바타), **관전/난입**(진행 중 입장 → 다음 판 합류), **초대 링크**(`?room=코드`)
- 동점 시 **타이브레이커**(총점→상단합→요트), **AI 빠르게** 토글, 결과 **클립보드 복사**

## 로컬 실행
```bash
npm install
npm start
# http://localhost:3000
```

## 배포 (Render 등)
- Build Command: `npm install`
- Start Command: `node server.js`
- 정적 파일 + WebSocket을 한 서버에서 처리하므로 별도 설정 불필요

## 데스크톱 앱(투명 창, 선택)
```bash
npm install
npm run app   # electron 창으로 실행 (항상 위, 투명 배경)
```

## 구조
- `public/game-core.js` — 규칙·점수·AI·게임 진행 엔진 (서버와 브라우저가 **공유**하는 단일 소스)
- `server.js` — 정적 서버 + 방/로비 관리 + 방마다 엔진 1개를 돌리는 권위 서버
- `public/index.html` — 클라이언트(상태 스냅샷을 받아 렌더만 담당)
- `electron/` — 투명 데스크톱 래퍼

## 온라인이 안 될 때
- `file://`로 열면 온라인을 못 써요. 배포된 주소(또는 `npm start`)로 접속하세요.
- 무료 호스팅은 미사용 시 잠들어 첫 접속이 느릴 수 있어요(콜드 스타트).
